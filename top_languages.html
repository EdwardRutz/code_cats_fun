STILL WIDLEY USED

  • COBOL:

    • Used largely in the finance industry and with mainframes.

  • FORTRAN Programs

    • Abbreviation for "Formula Translator." FORTRAN is still being used because it's the most efficient language for complex mathematical calculations for large systems.

    • CHARMM (Molecular dynamics)
    • Code_Saturne (Computational fluid dynamics)
    • NEMO (Oceanography)
    • QUANTUMESPRESSO (Materials modeling)
    • SPECFEM3D (Seismic wave propagation)
    • WRF (Weather forecasting)

NOTES

C

  • The most influential language of all time.
  • Many languages are C based.
  • Pure structured language and non-objected oriented.
  • Low level languaged requiring some memory management
  • Strongly typed
  • Compiled
  • Skill: Intermediate to Highly Advanced
  • Used for everything, mostly advanced programs such as games, utlilities, embedded systems, operating systems.
  • Not typically used for desktop, web or mobile applications.
  • More Info:
    • www.gcc.gnu.org
    • Book, The C Programming Language by Brian W. Kernighan and Dennis M. Ritchie (known as “The K&R Book”), Amazon Link
