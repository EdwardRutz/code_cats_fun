# QUIZ Q&A ## CSS What does it mean when a layout is fluid? The page elements have relative width units, like percentages. What is the purpose of responsive web design? Responsive design allows for the same code to work on multiple screen resolutions. What is the role of media queries in responsive design? Media quieries allow page elements to be adjusted based on width of the page. What are the three parts of responsive web design? Fluid images, fluid grids, and media quieries.
