PROGRAMMING FUDNAMENTALS

PROGRAMMING BASICS

  • Compiled languages create a machine code executable from source code (C, C++ Objective-C).
  • Interpretive language relies on the platform to interpret and run the program (PHP, JavaScript).
  • Hybrid-Intermediate Approach: Compile as much as possible then send it to the platform which finishes interpreting the program (Java, C#, VB.Net, Python).

CORE PROGRAMMING SYNTAX

  • CASE SENSITIVE: JavaScript is case-sensitive.
  • VARIABLE NAMES: Variables are the containers or "buckets" that hold a value used by the program.
    • CASE SENSITIVE: Variables are case sensitive (e.g. "var X" is different from "var x").
    • JS Variable names cannot start with a number and need to be clear and descriptive.
    • USE VAR: It is a good practice to use "var" with the variable name even though JavaScript does not require it. If it can't find the variable name it will create it.
    • Combine variables on one line as shorthand (e.g. "var day, month, year;") or list on seperate lines.
    • JAVASCRIPT IS A WEAKLY-TYPED LANGUAGE: once you create a variable in JS, you can put any type of data in it. Unlike many programming languages, JavaScript variables are not type specific.
